# Geospatial Data Processing with PostgreSQL, PostGIS, GeoPandas, and Google Earth Engine

This repository contains foundational examples and workflows for working with geospatial data using **PostgreSQL**, **PostGIS**, and **GeoPandas**, with integration to **Google Earth Engine**.

The goal of this project is to demonstrate how modern geospatial data pipelines can combine **spatial databases**, **Python geospatial libraries**, and **cloud-based geospatial platforms** to process, analyze, and visualize spatial datasets efficiently.

This repository is useful for students and researchers learning **GIS data management, spatial analysis, and GeoAI workflows**.

### Description

**PostgreSQL & PostGIS**

* Introduction to relational database concepts
* Creating spatial tables
* Spatial indexing
* Basic and advanced spatial queries
* Working with vector geometries in SQL

**GeoPandas**

* Reading and writing geospatial datasets
* Spatial data manipulation
* Coordinate reference systems (CRS)
* Spatial joins and geospatial analysis
* Integration with Google Earth Engine data workflows
